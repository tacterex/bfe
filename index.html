Introduction

This language provides you a 255 element array of 2-byte signed integers. You can access those elements, change them, print them, and ask user to input them

>/< (move pointer)

With > and < operators, you can move move pointer right or left. Starting position of pointer is 0 (first element) Pointer shows which cell you access right now. For exaample, after following operators >>>>>><<><< pointer will move 2 cells right. If your po inter is hogher than 254, error will be raised

+/- (increment / decrement)

Using + and -, you can increment/decrement value in current cell. If ypur value will go out of range of 2 bytes, it will return to opposite 'border' of value, creating a loop. For example, next operators +>++>+++<- will set 1 cell to 1, second to 1 and 3 to 3

. (char print)

With . you can print value from current cell as a char. You will see char with ASCII code of this cell. For exaample, nex code +++++++++++. will print symbol with ASCII code of 11

% (number print)

With . you can print value from current cell as a number. You will see number that was stored in cell . For exaample, nex code +++++++++++. will print 11

! (set zero)

! operator will set all values in array to 0. For example, next code +++!++++% will print 4

, (user input)

, will ask user to enter number from console. You need to enter number (not char!) you want to set and press Enter. For example, next code >,>,>,.<.<. will ask user to input 3 numbers and print them in reverse order as a char

[/] (loops)

[ and ] are useful to create loops. [ mean loop begining. ] checks if acessed cell isn't zero and, if so, jumps to the corresponding [ bracket. For example, next code +++++++++[>++++++++<-]>. will write 'H' in console

Enjoy coding this useful language!
